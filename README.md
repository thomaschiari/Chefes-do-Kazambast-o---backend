# Chefes do Kazambastão
Repositório para o Back End em Spring Boot, do projeto Chefes do Kazambastão, para a disciplina de Arquitetura de Objetos e Times Ágeis do curso de Ciência da Computação do Insper.

### Arquiteturas utilizadas:
Para o projeto, utilizamos Java com Spring Boot, em uma API REST, com banco de dados H2. Também utilizamos o Swagger para documentação da API. Para o Front End, utilizamos ReactJS. Acesse o nosso repositório do Front End [aqui](https://github.com/thomaschiari/Chefes-do-Kazambastao-frontend)

### Como rodar o projeto:
Para rodar o projeto, é necessário ter o Java 11 instalado, e o Maven. Para rodar o projeto, basta rodar o comando `mvn spring-boot:run` na pasta raiz do projeto. Para acessar a documentação da API, basta acessar o link `http://localhost:8080/swagger-ui.html` no navegador após rodar o projeto. Para acessar o banco de dados, basta acessar o link `http://localhost:8080/h2-console` no navegador após rodar o projeto.

### Rotas:
Nossas rotas estão disponíveis tanto na documentação Swagger, quanto no arquivo [`ROTAS.md`](ROTAS.md) na raiz do projeto.
